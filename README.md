# VibeCoding 网页赛跑游戏

一个无需安装、无需联网、双击即可运行的单人网页赛跑小游戏。项目使用原生 HTML、CSS 与 JavaScript 编写，画面和赛车均通过 Canvas 实时绘制。

## 游戏特色

- 单人玩家与电脑赛车竞速
- 支持方向键和 WASD 操作
- 赛道障碍、碰撞减速与电脑自动换道
- 倒计时、实时速度、当前名次、比赛进度和计时
- 胜负结算及一键重新开始
- Claude Code 风格的暖色终端界面
- 无第三方依赖，可离线运行

## 快速开始

1. 下载项目中的 `index.html`。
2. 使用 Edge、Chrome 或其他现代浏览器打开该文件。
3. 等待三秒倒计时结束后开始比赛。

项目不需要安装 Node.js，也不需要启动本地服务器。

## 操作方式

| 操作 | 方向键 | WASD |
| --- | --- | --- |
| 加速 | `↑` | `W` |
| 刹车 | `↓` | `S` |
| 向左转向 | `←` | `A` |
| 向右转向 | `→` | `D` |

碰到路障或油渍会明显减速。玩家或电脑赛车率先抵达终点时，本局比赛结束。

## 技术实现

- HTML5 Canvas：绘制赛道、赛车、障碍和环境元素
- CSS3：终端风格界面与响应式布局
- JavaScript：游戏循环、键盘输入、碰撞检测和电脑对手逻辑

## 浏览器支持

推荐使用最新版 Microsoft Edge 或 Google Chrome。游戏以桌面键盘操作为主，同时对窄屏窗口进行了响应式适配。
